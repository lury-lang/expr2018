## Installation

### Ubuntu / Debian

We built for Ubuntu 18.04 LTS (Bionic Beaver).

```bash
$ sudo apt-get install antlr4
$ pip install -r src/requirements.txt
```

### OSX

```bash
$ brew install antlr
$ sudo -H pip install -r src/requirements.txt
```
